## 🎵 Como Começar em 3 Passos

### 1️⃣ Abra 2 Terminais

**Terminal 1:**
```bash
cd backend && npm install && npm run dev
```

**Terminal 2:**
```bash
cd frontend && npm install && npm run dev
```

### 2️⃣ Abra o Navegador

👉 **http://localhost:5173**

### 3️⃣ Use a App!

Pronto! Sistema de Louvores totalmente funcional! 🎉
